# vue3-core


&emsp;使用esbuild.js构建，运行：
`pnpm i`
`pnpm run dev`

&emsp;用最小实现方式实现vue3，仅供个人深入学习使用，目前已实现:

* reactive
* shallowReactive
* readonlyReactive
* ref
* component
* watch
* watchEffect
* getInstance
* 组件渲染
* diff更新
* props状态更新
* emits组件自定义事件
* solt组件插槽
* onMounted等生命周期钩子函数

**支持 optionsApi、componentApi，目前仅支持渲染虚拟节点，模板语法正在更新中...**
